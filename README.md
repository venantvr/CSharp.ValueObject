# CSharp.ValueObject

Implémentation du pattern Value Object (DDD) en C# : objets immuables comparés par valeur plutôt que par référence.

## Structure

- `ValueObject/Adresse.cs` — Exemple de Value Object (adresse)
- `ValueObject/UnitTest.cs` — Tests d'égalité et de comparaison
